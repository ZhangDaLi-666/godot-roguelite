# Conversation Notes

This folder keeps short handoff notes for important Codex learning sessions.

The goal is not to store full chat transcripts. The goal is to preserve the parts that help future conversations continue smoothly:

- What was decided.
- What was built or changed.
- What the user learned.
- Which commands mattered.
- What should happen next.

## Naming

Use this format:

```text
YYYY-MM-DD-session-NNN-short-topic.md
```

Example:

```text
2026-07-01-session-001-project-bootstrap.md
```

## Suggested Structure

```markdown
# Session N: Short Title

## Summary

## Decisions

## Commands And Concepts

## Files Changed

## Current Git State

## Next Steps
```

Keep each note concise. A useful summary is better than a long transcript.
